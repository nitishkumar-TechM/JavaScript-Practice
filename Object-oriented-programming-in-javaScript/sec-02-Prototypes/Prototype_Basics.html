Prototypes and Prototypical Inheritance

In JavaScript, every object has a prototype.
A prototype is also an object.
Every Object has a prototype object excepts the Root object.
All JavaScript objects inherit their properties and methods from their prototype.
The prototype of an object can be accessed using the
__proto__ property or the Object.getPrototypeOf() method.
eg.- let x = {};
Object.getPrototypeOf(x);

A prototype is essentially a parent of another object.
Every Object in javaScript accepts only a single object.

The prototype chain is a series of links between objects that allows JavaScript to look up properties and methods.
When you try to access a property or method on an object, JavaScript first checks if the object has that property or method.
If it doesn't, it looks up the prototype chain until it finds it or reaches the end of the chain.
So, this called prototypical inheritance in action.
